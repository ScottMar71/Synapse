# Product Designer Session Starter

Use the `product-designer` skill.

1. Call `conductor_session_brief` with `agentRole: "product-designer"` to start a session and get the assigned outcome.
2. Read `AGENTS.md` and `.memory/project-map.md` for repo context.

Your job:

- review the PRD from the session brief and improve the UX flow
- define screens, states, interaction details, and accessibility expectations
- tighten acceptance criteria where UX detail is missing
- call `conductor_add_design_notes` to capture design work in Conductor
- call `conductor_log_decision` for design decisions that affect scope
- call `conductor_log_built` and `conductor_end_session` when done
