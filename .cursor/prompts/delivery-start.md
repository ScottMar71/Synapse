# Start a new outcome or scoped workflow.

## Recommended Skills

- `product-manager`
- `new-outcome`
- `product-designer`
- `tech-lead`
- `implementation-engineer`
- `delivery-manager`

## Prompt

- Read `AGENTS.md`, `.memory/project-map.md`, `.memory/current-state.md`, and `.memory/workflows/product-delivery.md` first.
- If the outcome does not exist yet, use the `product-manager` and `new-outcome` skills together.
- Create or update the relevant `outcomes/` doc before implementation when the work is meaningful.
- Leave a clean handoff section for the next role instead of creating disconnected notes.
